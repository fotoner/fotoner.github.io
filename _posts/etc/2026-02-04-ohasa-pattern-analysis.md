# 오하아사 별자리 운세의 비밀을 파헤쳐봤습니다

매일 아침 일본 후지TV의 "메자마시 테레비(めざましテレビ)"를 보면 나오는 그 코너, 오하아사(おはアサ) 별자리 운세. "오늘의 1위는 사자자리!" 하면서 12개 별자리 순위를 쫙 보여주는데요.

문득 궁금해졌습니다. **이 순위는 대체 어떤 기준으로 정해지는 걸까?**

저는 현재 **운세 앱**을 개발하고 있습니다. "오늘 가챠 운은 어떨까?", "티켓팅 성공할 수 있을까?" 같은 맞춤형 운세를 제공하는 서비스인데요. 개발 과정에서 기존 운세 서비스들이 어떤 로직을 사용하는지 궁금해졌고, 가장 유명한 오하아사를 분석해보기로 했습니다.

---

## 배경지식: 별자리 운세는 어떻게 작동하나?

본격적인 분석에 앞서, 점성술의 기본 개념을 간단히 정리해볼게요.

### 12개 별자리와 4원소

서양 점성술에서 12개 별자리는 **4가지 원소(Element)**로 나뉩니다:

| 원소      | 별자리                         | 성질                   |
| --------- | ------------------------------ | ---------------------- |
| 불(Fire)  | 양자리, 사자자리, 사수자리     | 열정적, 행동적, 리더십 |
| 흙(Earth) | 황소자리, 처녀자리, 염소자리   | 현실적, 안정적, 신중함 |
| 바람(Air) | 쌍둥이자리, 천칭자리, 물병자리 | 지적, 사교적, 소통     |
| 물(Water) | 게자리, 전갈자리, 물고기자리   | 감성적, 직관적, 공감   |

### 양극성(Polarity)

12개 별자리는 또한 **양극성**으로도 분류됩니다:

| 극성                     | 원소     | 성질                   |
| ------------------------ | -------- | ---------------------- |
| 양성(Positive/Masculine) | 불, 바람 | 외향적, 능동적, 표현적 |
| 음성(Negative/Feminine)  | 물, 흙   | 내향적, 수용적, 반응적 |

이건 좋고 나쁨이 아니라 에너지의 방향성을 의미해요.

### 일일 운세의 원리

전통 점성술에서 일일 운세는 주로 **달(Moon)**의 위치를 기준으로 합니다. 달은 약 2.5일마다 별자리를 옮겨가며, 그날의 "감정적 분위기"를 결정한다고 봅니다.

여기에 **태양, 행성들의 위치와 각도(애스펙트)**를 종합해서 각 별자리의 운세를 계산하죠.

그렇다면 오하아사도 이런 전통적인 방식을 따를까요?

## 데이터 수집: 79일간의 오하아사 순위

앱 개발을 위해 오하아사 순위 데이터를 직접 수집했습니다.

2025년 6월부터 2026년 2월까지, 총 **79일간의 데이터**입니다.

그리고 같은 날짜의 **실제 천문 데이터**—달의 위치, 행성 배치 등—를 NASA JPL 데이터로 계산해서 비교했습니다. (저희 앱은 NASA의 DE440s ephemeris 데이터와 Skyfield 라이브러리를 사용해 실제 행성 위치를 계산합니다.)

결과는... 예상보다 훨씬 흥미로웠습니다.

## 발견 1: 같은 원소끼리 뭉친다 (하지만 항상은 아니다)

첫 번째로 눈에 띈 패턴이 있었습니다.

**같은 원소의 별자리 3개가 항상 붙어다닙니다.**

오하아사 순위를 보면, 같은 원소의 3개 별자리가 거의 항상 연속으로 나옵니다.

예를 들어 2026년 1월 26일 순위:

```
1위 염소자리  ┐
2위 황소자리  ├─ 흙(Earth)
3위 처녀자리  ┘
4위 전갈자리  ┐
5위 물고기자리 ├─ 물(Water)
6위 게자리    ┘
7위 물병자리  ┐
8위 쌍둥이자리 ├─ 바람(Air)
9위 천칭자리  ┘
10위 사수자리 ┐
11위 양자리   ├─ 불(Fire)
12위 사자자리 ┘
```

### 상위권 vs 하위권: 다른 규칙이 적용된다

여기서 흥미로운 차이를 발견했습니다.

| 구간            | 완벽 그룹핑 비율 |
| --------------- | ---------------- |
| 상위권 (1-6위)  | **100%**         |
| 하위권 (7-12위) | **60.8%**        |

**상위권(1-6위)은 79일 전부 완벽한 원소 그룹핑**을 보여줍니다. 1-2-3위가 같은 원소, 4-5-6위가 같은 원소.

하지만 **하위권(7-12위)은 39.2%에서 그룹핑이 깨집니다.**

### 하위권 브레이크 패턴 분석

브레이크가 발생할 때, **무작위가 아니라 딱 3가지 패턴만** 나타납니다.

하위권 6개를 2개씩 쌍으로 묶어서 분석했습니다 (7-8위 / 9-10위 / 11-12위):

- **A** = 같은 원소 쌍
- **B** = 다른 원소 쌍

참고: 완벽 그룹핑(7-8-9 동일 + 10-11-12 동일)도 ABA 패턴이지만, 여기서는 **브레이크가 발생한 31일만** 분석합니다.

| 패턴 | 구조                            | 비율    | 특징          |
| ---- | ------------------------------- | ------- | ------------- |
| ABA  | 7-8 유지, 9-10 혼합, 11-12 유지 | **42%** | 중간에서 섞임 |
| BAA  | 7-8 혼합, 9-10 유지, 11-12 유지 | **29%** | 시작부터 섞임 |
| AAB  | 7-8 유지, 9-10 유지, 11-12 혼합 | **29%** | 끝에서 섞임   |

**핵심 발견:**

- **7-8위 같은 원소**: 71% (ABA + AAB 패턴)
- **11-12위 같은 원소**: 71% (ABA + BAA 패턴)
- **중간(9-10위)에서 섞임이 가장 흔함**: ABA 패턴이 42%로 최다

예시 - 2025년 10월 16일 (BAA 패턴):

```
7위 처녀자리  ┐ 흙  ┐
8위 게자리    ┘ 물  ┘ B (다른 원소)
9위 염소자리  ┐ 흙  ┐
10위 황소자리 ┘ 흙  ┘ A (같은 원소)
11위 전갈자리 ┐ 물  ┐
12위 물고기자리┘ 물  ┘ A (같은 원소)
```

**중요한 점 두 가지:**

1. **브레이크 ≠ 완전 랜덤**: 그룹핑이 깨져도 **부분적 그룹핑 경향**은 유지됩니다.

   - 7-8위 같은 원소: 71%
   - 11-12위 같은 원소: 71%
   - 즉, 양 끝(7-8, 11-12)은 그룹핑을 유지하고, **중간(9-10)에서만 섞이는** 경향

2. **양극성 3:3 배분은 100% 유지**: 하위권 6개 중 양성 3개, 음성 3개는 항상 지켜집니다. 브레이크가 발생해도 이 규칙은 절대 깨지지 않습니다.

## 발견 2: 양극성(Polarity) - 진짜 핵심 패턴

여기서 더 중요한 발견이 있습니다.

오하아사 순위를 양극성 관점으로 보면 **완벽한 패턴**이 나타납니다:

```
[양성이 상위인 날]
불 > 바람 > 물 > 흙  또는
바람 > 불 > 물 > 흙

[음성이 상위인 날]
물 > 흙 > 불 > 바람  또는
흙 > 물 > 불 > 바람
```

79일 데이터를 분석한 결과:

| 1-2위 원소 조합 | 횟수 | 비율 |
| --------------- | ---- | ---- |
| 물+흙 (음성)    | 40회 | 51%  |
| 불+바람 (양성)  | 39회 | 49%  |

**같은 극성의 원소가 항상 함께 상위권 또는 하위권을 차지합니다!**

이건 우연이 아닙니다. 79일 중 79일, **100%** 이 패턴을 따릅니다.

## 발견 3: 달 위치와의 상관관계는 약하다

"별자리 운세니까 당연히 달 위치를 보겠지?"라고 생각했는데요.

저희가 계산한 실제 달 위치와 비교해봤습니다.

**달의 원소와 오하아사 1위 원소가 일치한 날은 79일 중 약 21일(27%)뿐이었습니다.**

특히 2026년 1월 28일이 결정적이었어요.

이 날 달은 황소자리(흙)에 있었고, **불(Fire) 별자리에는 행성이 하나도 없었습니다**. 태양, 수성, 금성, 화성... 전부 다른 원소 별자리에 있었죠.

그런데 오하아사는 불(Fire)을 1위로 뽑았습니다.

**실제 천문 현상과 오하아사 순위 사이에는 직접적인 연관이 없어 보입니다.**

## 발견 5: 연속일 패턴 유지 경향

하위권(7-12위)의 원소 배열 패턴을 날짜별로 추적해봤더니, **연속일에 같은 패턴이 유지되는 경향**이 있었습니다.

```
2025-10-27 (월): WWWEEE ← 물물물흙흙흙
2025-10-28 (화): WWWEEE ← 같은 패턴!
2025-10-29 (수): WWWEEE ← 3일 연속!
2025-10-30 (목): FFAAFA ← 변경
2025-10-31 (금): FFAAFA ← 같은 패턴!
```

분석 결과:

- **39.7%** 확률로 전날과 같은 하위권 패턴 유지
- 최대 **3일 연속** 같은 패턴 관찰
- 행성 이동이 점진적이라 패턴이 "끈적하게" 유지되는 것으로 추정

이는 오하아사가 **천문 현상의 점진적 변화를 반영**하거나, 또는 **의도적으로 연속성을 부여**하는 것일 수 있습니다.

## 결론: 오하아사의 알고리즘

분석 결과를 종합하면:

### 확실한 것

1. **양극성(Polarity) 기반**: 음성(물/흙) vs 양성(불/바람)으로 양분 (100%)
2. **상위권 완벽 그룹핑**: 1-6위는 100% 원소 그룹핑
3. **하위권 부분 그룹핑**: 7-12위는 60.8% 완벽, 39.2% 부분 브레이크
4. **2-3일 주기 순환**: 1위 원소가 2-3일마다 변경
5. **요일별 패턴**: 목요일 양성 우세(73%), 화요일 음성 우세(69%)

### 불확실한 것

1. **무엇이 극성을 결정하는가?**: 달 위치와 27%만 일치
2. **원소 내 순서**: 같은 원소 내에서 1-2-3위 결정 기준
3. **하위권 브레이크 기준**: 왜 목/금요일에 브레이크가 더 많은가?

### 발견: 원소 전환 패턴

79일 데이터에서 1위 원소가 바뀔 때의 전환 패턴:

```
바람(AIR) → 흙(EARTH): 8회
흙(EARTH) → 불(FIRE): 6회
물(WATER) → 바람(AIR): 6회
불(FIRE) → 물(WATER): 5회
```

순환 경향: `바람 → 흙 → 불 → 물 → 바람`

### 추정 알고리즘

오하아사는 다음 로직을 사용하는 것으로 추정됩니다:

```
1. 요일 기반 극성 편향
   - 목요일: 양성(불/바람) 73% 확률
   - 화요일: 음성(물/흙) 69% 확률

2. 상위권(1-6위) 규칙
   - 100% 완벽한 원소 그룹핑 (1-2-3위 같은 원소, 4-5-6위 같은 원소)
   - 양극성에 따라 어느 원소가 상위인지 결정

3. 하위권(7-12위) 규칙
   - 60.8% 완벽 그룹핑, 39.2% 부분 브레이크
   - 브레이크 시 딱 3가지 패턴만 사용 (ABA 42%, BAA 29%, AAB 29%)
   - 7-8위, 11-12위는 같은 원소 유지 경향 (71%)
   - 양극성 3:3 배분은 100% 유지

4. 2-3일 연속 유지
   - 같은 원소가 2-3일 연속 1위
   - 하위권 패턴도 39.7% 확률로 다음날 유지

5. 전환 시 순환 패턴
   - 바람 → 흙 → 불 → 물 → 바람
```

## 우리가 배운 것, 그리고 앱에 적용할 것

**실제 천문 현상과 오하아사 순위는 직접적인 연관이 없어 보입니다.**

하지만 오하아사는 **점성술의 양극성(Polarity) 개념**을 일관되게 적용하고 있었습니다. 단순히 랜덤이 아니라, 나름의 규칙이 있는 거죠.

### 오하아사 분석에서 배운 핵심

1. **상위권은 확실하게**: 1-6위는 100% 완벽한 그룹핑으로 시청자에게 명확한 메시지 전달
2. **하위권은 유연하게**: 7-12위는 때때로 그룹핑을 깨서 변화감 부여
3. **양극성은 절대적**: 양성/음성 6:6 배분은 어떤 상황에서도 100% 유지
4. **요일별 캐릭터**: 목요일은 양성(활동적), 화요일은 음성(안정적)이라는 요일 특성 부여

### 저희 앱에 적용할 것

1. **양극성 그룹핑 100% 적용**: 오하아사의 가장 확실한 패턴 채용
2. **상위권 완벽 그룹핑**: 1-6위는 원소별 3연속 배치
3. **요일별 극성 편향 반영**: 목요일 양성, 화요일 음성 경향 적용
4. **실제 천문 데이터 기반**: 오하아사와 달리, 실제 달/행성 위치로 극성 결정

오하아사의 "왜 오늘은 양성이고 내일은 음성인가?"에 대한 답은 찾지 못했지만, 저희는 **실제 행성 배치**를 기준으로 이를 결정할 계획입니다.
